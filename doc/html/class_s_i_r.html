Detailed Description

template<class Particle, class Viewer>
class SIR< Particle, Viewer >

SIR algorithm.

This implementation is inspired from the paper or Isard and al. "conditionnal density propagation for visual tracking" It is actually the Condensation version of the SIR algorithm.
