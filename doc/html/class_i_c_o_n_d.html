Detailed Description

template<class Particle, class Detector, class Viewer>
class ICOND< Particle, Detector, Viewer >

ICONDensation algorithm.

This implementation is inspired from the paper or Isard and al. "unifying low-level and high-level tracking in a stochastic framework"
